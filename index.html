🚀 Quick Setup Guide

1
Prepare Looker Studio Reports
Make sure your reports are published and set to "Anyone with the link can view"
2
Get Embed URLs
In each report, click Share → Embed → Copy the embed URL
3
Replace Placeholder URLs
Find the commented iframe code and replace "YOUR_LOOKER_STUDIO_URL_HERE"
4
Remove Placeholders
Delete the placeholder divs after adding your real Looker Studio embeds
